Using the due date, last period date, ultrasound date, conception date, or IVF transfer date, this calculator generates an expected pregnancy timeline.

[https://www.calculator.io/pregnancy-calculator/](https://www.calculator.io/pregnancy-calculator/)

== Usage ==

Add the Pregnancy Calculator shortcode to your page, post or sidebar:

`[ci_pregnancy_calculator]`

Add the following code to your template where you would like the Pregnancy Calculator to appear:

`<?php display_ci_pregnancy_calculator(); ?>`

1. Upload the Pregnancy Calculator /ci_pregnancy_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Pregnancy Calculator plugin through the "Plugins" menu in WordPress.
